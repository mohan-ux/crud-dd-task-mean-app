# Screenshots Directory

Place your screenshots in this directory. Required screenshots:

1. **github-actions-workflow.png** - GitHub Actions workflow configuration and successful execution
2. **docker-hub-images.png** - Docker images pushed to Docker Hub
3. **docker-containers.png** - Docker containers running on the VM
4. **application-ui.png** - MEAN Stack CRUD application running in browser
5. **nginx-setup.png** - Nginx reverse proxy configuration

## How to Capture Screenshots

### Windows
- Press `Win + Shift + S` to use Snipping Tool
- Or press `PrtScn` to capture full screen

### Linux
- Use `gnome-screenshot` or `flameshot`
- Press `PrtScn` for full screen

### macOS
- Press `Cmd + Shift + 4` for selection
- Press `Cmd + Shift + 3` for full screen
